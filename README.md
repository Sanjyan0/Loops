# JavaScript Loops Assignment

## Objective
A JavaScript program that utilizes control structures (for and while loops) to perform mathematical operations on variables. 

## Features
- Calculates the sum of the first `n` numbers.
- Prints the multiplication table for a variable `n`.
- Takes a specific number (e.g., 153) and computes:
  - The sum of its digits.
  - Whether it is an Armstrong number.
  - Whether it is a prime number.
  - All of its factors.

## How to Test
1. Open `index.html` in your web browser.
2. Open the Developer Tools (Right-click > Inspect).
3. Navigate to the **Console** tab to view the printed results.
